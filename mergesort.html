Mergesort

Mergesort is an efficient, general purpose, comparison-based sorting algorithm that uses the concepts of divide and conquer and recursion to sort an array. It was invented by John von Neumann in 1945.

Mergesort runs in a guaranteed O(n log n) time, which is significantly faster than the average and worst-case running times of several other sorting algorithms. Most implementations produce a stable sort, which means that the implementation preserves the input order of equal elements in the sorted output.

  1. Divide by finding the number mid of the position midway between low and high. This is exactly the same as finding the midway in binary search: add low and high, divide by 2, and round down.

    The orange bars represents the low and high values and shows the boundary of the current subarray. For clarity the mid value is also shown as a red bar.

  2. Conquer by recursively sorting the subarrays in each of the two subproblems created by the divide step. That is, recursively sort the subarray array[low..mid] and recursively sort the subarray array[mid+1..high].

    Comparisons when sorting are shown as green bars.

  3. Combine by merging the two sorted subarrays back into the single sorted subarray array[low..high].

    The blue bar represents the current element that is merged back into the source array.

Welcome to Adavis

This website was created as a group assignment for the course Webbapplikationer för mobila enheter, Malmö Högskola, spring 2018.

It provides pedagogical and pleasant visualisation of several sorting and search algorithms. It is implementing the D3 JavaScript library, used for data visualisation.
