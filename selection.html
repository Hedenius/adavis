Selection sort

Selection sort is one of the most simple sorting algorithms, as well as one with quite poor performance on large amounts of data. It has a time complexity of O(n2) but due to sorting in-place it can still be useful if space is very limited.

The algorithm will for each element in the array compare it against the rest. In the animation above the element currently being compared against the rest can be seen as the blue bar. Each blue bar acts like a starting point for a nestled for loop iterating over the remaining right side of the list. The current index of the loop can be seen as the bar marked green.

When the inner loop, the green bar, has found an element that is smaller than the one currently being compared to the rest this element will be marked in orange. Whichever element is marked orange (the smallet) by the end of the inner loop will be swapped with the element it was compared to.

This is repeated until the outer loop reaches the end and no sorting is left to be done.
