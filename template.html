Welcome to Adavis

This website was created as a group assignment for the course Webbapplikationer för mobila enheter, Malmö Högskola, spring 2018.

It provides pedagogical and pleasant visualisation of several sorting and search algorithms. It is implementing the D3 JavaScript library, used for data visualisation.

Created by students Viktor Ekström, Alena Hramyka and Martin Winqvist.
